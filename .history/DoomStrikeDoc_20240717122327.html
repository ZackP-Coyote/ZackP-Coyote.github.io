DoomStrike Documentation

Project Description

DoomStrike is an online third-person shooter with characters having unique traits giving the player the opportunity
to play at their own strengths in shooter games involving different guns with their own specific loadouts. “DoomStrike”
sets itself apart from other shooters because there are not many competitive third-person shooters that offer a
different play-style and abilities for each weapon.


DoomStrike is still in Pre-Alpha version and still undergoing development



Project Features

  • Users can choose from a variety of loadouts
  • Users can select different maps for a unique experience
  • Users can either play against AI or online (work in progress)

Project Requirements

  • Users must have minimum computer specifications to run Unreal Engine 5.2
  • Users must have an internet connection in order to download

Project Installation

    DoomStrike is a downloadable asset through the provided link in the GitHub link below

  1. Users must clone the Git repository
  2. Users must download the Unreal Engine 5.2 or alter version
  3. Users must open the project in Unreal Engine for DoomStrike to launch

Project Usage

Users can access DoomStrike by visiting the following URL: https://github.com/ZackP-Coyote/DoomStrike

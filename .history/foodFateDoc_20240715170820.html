FoodFate Documentation

Project Description

FoodFate is a web application designed to assist users in finding dinner recommendations within a specified area.
Users can enter their desired location, set parameters such as distance and food preferences to receive three random restaurant suggestions.

Project Features

  • Users can enter their desired location
  • Users can set parameters such as distance and food preferences
  • Users can receive three random restaurant suggestions

Project Requirements

  • Users must have a web browser
  • Users must have an internet connection

Project Installation

FoodFate is a web application and does not require installation.

Project Usage

Users can access FoodFate by visiting the following URL: https://zackp-coyote.github.io/Food_Fate/

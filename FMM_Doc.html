Find My Microwave Documentation

Project Description

Find My Microwave is a mobile android app that helps California State University students and faculty locate useful campus resources.

Project Features

  • Users can use their android phone to locate Microwaves on campus
  • Users can look up information about the microwave including which floor it is on and if it is available

Project Requirements

  • Users must have an Android phone
  • Users must have an internet connection

Project Installation

  1. Download the GitHub repository
  2. Open the project in Android Studio
  3. Run the project on an Android phone

Project Usage

Users can access Find My Microwave by visiting the following URL: https://github.com/ZackP-Coyote/findmymicrowave
